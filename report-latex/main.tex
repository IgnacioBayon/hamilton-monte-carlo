% ---- Current Approaches + Limitations ----
\begin{frame}{Current Approaches in DMTA}
\begin{itemize}
    \item Standard workflow in the field:
    \begin{itemize}
        \item Statistical tests: ANOVA, Kruskal-Wallis $+$ post-hoc comparisons
        \item PCA for exploratory visualization
    \end{itemize}
    \item These techniques have been applied \textbf{for years without major updates}.
    \item More advanced methods (advanced Statistics, Machine Learning) are rarely used:
    \begin{itemize}
        \item Data are \textbf{small, noisy, multicollinear} $\Rightarrow$ challenging
        \item Researchers rely on familiar statistics
    \end{itemize}
\end{itemize}

\begin{block}{Key message}
Applying and validating \textbf{modern probabilistic models} on such a complex, reduced dataset
is itself a significant contribution.
\end{block}
\end{frame}
